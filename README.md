# Lifx CLI

```
$ ./lifx-cli.js --help
lifx-cli.js [command]

Commands:
  lifx-cli.js color <hue> <saturation>      Set light color
  <brightness> <kelvin>
  lifx-cli.js power <on|off>                Power light on or off
  lifx-cli.js discover                      Discover lights on your network

Options:
  --version  Show version number                                       [boolean]
  --help     Show help                                                 [boolean]
```
